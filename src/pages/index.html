Project Evergreen's goal is to help everyone succeed in making a modern website or application using the latest in web standards and technologies. With the trend of browsers going "evergreen" and making the web a constantly maturing platform, Project Evergreen sees an opportunity to advocate for "evergreen" sites and apps, and we think that's pretty cool!

Project Evergreen's motivations are rooted in the fundamental belief that when websites is written with standards in mind, we can maximize the outcomes for all involved. Making websites shouldn't be complicated, and should be easy to get started with. Let's make the web (and web dev) a better experience for everyone! ✌️
