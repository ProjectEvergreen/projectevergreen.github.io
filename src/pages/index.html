Project Evergreen's goal is to help promote modern web standards and sustainability through a focus on practical resources for developers.



With the rise of evergreen browsers, we see an opportunity for "evergreen" sites and app, and we think that's pretty cool!

Project Evergreen's motivations are rooted in the fundamental belief that when the web is written with standards in mind, we can maximize the outcomes for all involved. Making websites shouldn't be complicated, and should (still) be fun and easy to get started with. Let's make the web a better place for everyone!
